# 논산시 로코나19 현황 및 소비 동향 대시보드

### 데이터 
- 논산시 재난안전대책본부 일일상황보고 (코로나 확진자수, 격리자수, 검사자수 등)
- 충청남도 코로나19 일일상황보고 (확진자 수)
- 신한카드 매출 데이터
- 논산시 행정동 shp
- 충청남도 시군구 shp

### 사용한 툴
- Tableau
- Python
- QGIS
