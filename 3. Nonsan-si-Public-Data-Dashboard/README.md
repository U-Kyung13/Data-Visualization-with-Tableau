# 논산시 공공데이터 활용 동적 대시보드
### 데이터 
- 논산시 공공데이터 
  - 의료기관 정보(위치, 병원시설 종류, 전문의 진료과목, 소유하고 있는 의료장비 종류 등)
  - 노인인구비율 및 인구수
  - 어린이보호구역 위치 정보
  - CCTV 정보(위치, 화소, 촬영방면, 설치경과개월 등)
- 논산시 행정동 shp

### 사용한 툴
- Tableau
- Python
- QGIS
